# 概要
「Muscle + 」

筋トレを長く続けられるように、楽しく筋トレの記録をするサービスです。
# URL
~~https://muscle-plus.com~~ (お金ないので、ドメイン取るのやめました)

右上のメニューバーからログインおよび新規登録が行えます。
また、【かんたんログイン】ボタンからゲストユーザーとして、ログインすることができます。

# 機能一覧
- ユーザー登録・ログイン機能
- Facebookログイン機能
- 筋トレ記録機能
- フォロー機能
- いいね機能
- コメント機能
